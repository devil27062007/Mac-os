# Mac-Os

A browser-based Mac OS style desktop experience built with HTML, CSS, and JavaScript.
It is making inspired by the vintage Mac OS.

## Overview

This project recreates a classic desktop environment with a retro UI, interactive icons, draggable windows, and a developer profile section.

## Features

- Login screen flow
- Desktop-style navigation bar with File, View, and Developer menus
- Draggable desktop icons and windows
- System and Trash folder behavior
- Create, open, rename, delete, and restore file-like items
- Print modal simulation
- Developer profile window with About, Projects, Skills, and Contact tabs

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- system.css
- Tailwind CSS (CDN)

## Project Structure

- index.html: Login page
- main.html: Desktop interface
- index.css: Custom styling
- script.js: Main interaction logic
- assets/: Icons, backgrounds, loading media, and sounds

## Getting Started

1. Clone or download this repository.
2. Open main.html in a browser to view the desktop directly.
3. To follow the full flow, start with index.html and log in.
